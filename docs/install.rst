Install
-------

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install tenda-everest

From release page
^^^^^^^^^^^^^^^^^

Download source code from `GitHub Releases page <https://github.com/quillcraftsman/tenda-everest/releases>`_

Clone from GitHub
^^^^^^^^^^^^^^^^^

.. code-block:: bash

   git clone https://github.com/quillcraftsman/tenda-everest.git
   make install
